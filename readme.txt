# CPP PROGRAM TO IMPLEMENT PRIORITY QUEUE USING HEAP SORT
	file name :- priority_queue_using_max_heap.cpp (For max priority queue).
	file name :- priority_queue_using_min_heap.cpp (For min priority queue).

## Compiling And Running

compile:	g++ priority_queue_using_max_heap.cpp (for max priority queue).
		g++ priority_queue_using_min_heap.cpp (For min priority queue).
run    :	./a.out

### Functions in program
	1- Insert :
		To insert an element in priority queue.
		Input : Elements priority.
	2- Remove :
		To delete an element from priority queue.
		Input : Index of an element to delete.
	3- Change Priority :
		To change the priority of an element.
		Input : New priority and index of an element.
	4- Extract Maximum Priority Element (For max priority queue):
		To get the highest priority element from queue.
	   Extract Minimum priority Element (For max priority queue):
		To get the lowest priority element from queue.
	5- Print The Heap :
		To print all the element in heap.
	6- Build Heap :
		To build the heap from given elements.
		Input : Array.
	7- Heapify :
		To maintain the heap property.(either max or min. depends on queue).
		Input : Array and index
	8- getParent :
		To get the index of parent element of given element.
		Input : index of an element.
	9- getLeftChild :
		To get the index of left child of given element.
		Input : index of an element.
	10- getRightChid :
		To get the index of right child of given element.
		Input : index of an element.
	
	The program is menu driven program. You have to choose the operation you want and
	input accoring to the selected option.
